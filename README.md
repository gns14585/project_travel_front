롤랜드 프로젝트 프론트엔드

기획의도
이번 프로젝트의 목적은 게임 및 컴퓨터 장비에 특화된 다양한 제품을 쉽게 찾고 구매할 수 있는 사용자 친화적인 쇼핑몰 구축과, 
사용자들이 자유롭게 의견을 교환하고 자신들의 경험과 지식을 공유 하는 플랫폼 제공, 사용자 친화적인 디자인과 높은 기술 완성도를 목표로 프로젝트를 진행 하게 되었습니다.
